SKILLS

I'm a curious problem solver. I build websites and fullstack web applications. I use MERN stack, AWS, SQL, JavaScript, SASS, HTML and CSS. I highlighted my key technical skills that I will bring to your project.

Front End Development

I use HTML, CSS and JavaScript to build attractive and responsive user interfaces for websites. During development I'm always thinking about best practices, SEO, accessibility and performance.

React Development

React is an open-source, front end, JS library for building UIs or UI components. It offers graceful solutions to some of front-end programming's most persistent issues, allowing me to build dynamic and interactive web applications with ease.

Full Stack Web Development

I use MongoDB Atlas, Express.js, React.js and Node.js to build web applications. The "MERN Stack Front To Back" is one of the most popular JavaScript stacks for building modern single-page web applications.

Database Development

I'm comfortable in working with SQL databases and at the same time I'm studying NoSQL databases. Creating any kind of application and software is impossible for any organization without the database.

Source Control

I use Git to track changes in my projects and maintain data integrity, while GitHub allows me to manage my Git repositories and make source code open-source. It's especially useful for distributed workflows.

Cloud Computing

I utilize AWS in my own projects as it is the most popular and most mature cloud platform. Companies of all sizes and domains are now shifting their environments into Cloud for cost-saving and better scalability.
